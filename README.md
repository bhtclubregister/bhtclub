# Demo Club Website - Clone

This is a demo website created based on the design and layout of the BHT Club website. This is for **design and educational purposes only**.

## Features

- Modern, colorful design with gradient backgrounds
- Responsive layout for mobile and desktop
- Interactive modals for login and registration
- Smooth scrolling navigation
- FAQ accordion functionality
- Animated elements on scroll
- Mobile-friendly menu

## Files Structure

```
bht-clone-website/
├── index.html      # Main HTML file
├── styles.css       # All styling and design
├── script.js        # Interactive functionality
└── README.md        # This file
```

## How to Use

1. Open `index.html` in a web browser
2. No server setup required - works directly in the browser
3. All functionality is client-side only

## Sections Included

- Header with navigation
- Hero section with game cards
- Event announcements
- Rewards section
- Advantages/Features section
- Registration guide
- Agent program section
- Blog posts section
- FAQ section
- Footer with links

## Design Features

- Gradient backgrounds
- Card-based layouts
- Hover effects and animations
- Modal popups
- Responsive grid layouts
- Modern color scheme (purple, orange, gold accents)

## Browser Compatibility

Works on all modern browsers:

This is a **demo website** for design purposes only. All forms are disabled and no actual functionality is implemented. This should not be used for any real gambling or casino operations.
